# Gemini 95

A nostalgic Windows 95 simulator built with modern web technologies, featuring AI-powered interactions through Google Gemini.

## Description

Gemini 95 is a web-based recreation of the classic Windows 95 operating system interface, enhanced with AI capabilities. Experience the retro UI while interacting with modern AI features through various integrated applications.

## Features

- **Classic Windows 95 Interface**: Authentic desktop environment with taskbar, start menu, and window management
- **Integrated Applications**:
  - **GemNotes**: AI-powered notepad for text editing and processing
  - **GemPaint**: Drawing application with AI art critique
  - **GemSweeper**: Classic Minesweeper game
  - **GemPlayer**: YouTube media player
  - **Gemini App**: Direct AI chat interface
  - **Chrome Browser**: Web browsing capabilities
  - **Doom II**: Retro gaming experience
- **AI-Powered Features**: Leverages Google Gemini AI for intelligent interactions
- **Responsive Design**: Modern web technologies with retro aesthetics

## Tech Stack

- **Frontend**: TypeScript, HTML5, CSS3
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **AI Integration**: Google Gemini AI (@google/genai)
- **Deployment**: Web-based (runs in browser)

## Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd gemini-95
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your Google Gemini API key:
   - Obtain an API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
   - Configure the API key in your environment or within the application

## Usage

### Development

Start the development server:
```bash
npm run dev
```

This will start the Vite development server, typically at `http://localhost:5173`.

### Build for Production

Build the project for production:
```bash
npm run build
```

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```
